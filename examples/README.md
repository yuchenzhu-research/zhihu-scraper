# Examples / 示例

此目录用于存放可以随仓库一起展示的示例抓取结果。

This directory stores sample exports that are intentionally kept in the repository.

边界约定：

- `data/` 只放本地运行产物
- `examples/` 只放用于展示的静态示例
- 不要把真实 Cookie、数据库或监控状态放进这里

Repository boundary:

- `data/` is reserved for local runtime artifacts
- `examples/` is reserved for static showcase samples
- Do not place real cookies, databases, or monitor state files here
